##基于SSH+easyUI+ActiveMQ+MySQL的校园宿舍管理系统。
分为系统管理员、楼宇管理员、学生三种权限。

##模块介绍：
###(1) 学生管理模块 
    此模块某些部分是对系统管理员、楼宇管理员和学生开放的。
###(2) 楼宇管理模块
    此模块只可以由系统管理员操作。
###(3) 楼宇管理员管理模块
   此模块只可以由系统管理员操作。
###(4) 宿舍管理模块
此模块是对系统管理员和楼宇管理员开放的。
###(5) 申请登记模块
  此模块只对学生用户开放。
###(6) 消息处理模块
   此模块只对楼宇管理员开放。
###(7) 日志查看模块
   此模块只可以由系统管理员操作。
